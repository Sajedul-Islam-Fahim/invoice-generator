# Invoice Generator Web

React frontend for Invoice Generator application.

## Tech Stack

- **Framework:** React 19
- **Build Tool:** Vite
- **HTTP Client:** Axios
- **Routing:** React Router DOM
- **State Management:** Zustand
- **Notifications:** React Hot Toast
- **Icons:** Lucide React

## Requirements

- Node.js 20+
- npm

## Installation

```bash
npm install
npm run dev
```

## Environment Variables

```env
VITE_API_URL=http://localhost:8000/api
```

## Pages

- `/` — Dashboard with stats
- `/clients` — Client management
- `/invoices` — Invoice list with filters
- `/invoices/create` — Create new invoice
- `/invoices/:id` — Invoice detail & PDF download
